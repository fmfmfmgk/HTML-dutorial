입력 후 전송 하면 ~~.jsp로 전송한다.
전송하기 위해서는 input 필드에 반드시 name속성이 있어야 한다.
title       : 마우스를 요소 위에 올려놓으면 메모박스가 실행
required    : 필수항목(반드시 입력해야할 경우)
autofocus   : 실행시 초기 커서 위치
pattern     : 정규식
placeholder : input필드에 나타나는 예시, 제목, 내용
readonly    : 값을 변경할 수 없는 필드, 반드시 value속성이 있어야 한다.
